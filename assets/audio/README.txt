ADD YOUR HOMECOMING SONG HERE
-----------------------------
Drop an audio file named exactly:

    homecoming.mp3

into this folder (house/assets/audio/homecoming.mp3).

When the speaker button (bottom-left) is switched on, that song will
play softly on a loop. A Hindi homecoming song such as
"Khat — Navjot Ahuja" works beautifully here.

Tips:
- Use an .mp3 file. Keep it under ~8 MB so it loads fast.
- Only add a song you have the right to use.
- If no homecoming.mp3 is present, a soft original chord pad plays
  instead, so there is always gentle sound.
